# Codex 문서 디렉토리

Codex가 이 저장소에서 수행한 별도 실행, 검증, 실험 기록을 모아두는 디렉토리.

## 문서 목록

- [2026-04-23_gpt_golden_truth_run.md](2026-04-23_gpt_golden_truth_run.md)
  - Track A / Track B GPT 시뮬레이션 실행 결과
  - 분리 저장 디렉토리와 산출물 경로
  - 비교 기준과 한계

## 원칙

- 기존 `agent/track_a/results_*`, `agent/track_b/results_*` 결과와 섞지 않는다.
- 실행 산출물은 별도 세션 디렉토리에 저장하고, 여기에는 해설과 요약만 남긴다.
- 문서에는 재현 가능한 명령, 모델, 비교 기준을 함께 적는다.
